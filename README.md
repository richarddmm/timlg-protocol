# TIMLG MVP - Smart Contracts

This repository contains the on-chain program (smart contracts) for **TIMLG MVP**, a time-locked betting protocol on Solana.

## Purpose
This repository is dedicated to the public hosting of the program's source code to facilitate **On-Chain Verification** on Solana explorers (Solscan, Solana.fm).

## Technical Stack
- **Framework**: Anchor 0.32.1
- **Language**: Rust
- **Cluster**: Devnet
